🔍 Clase 042 - LIKE y CONTAINS

Búsquedas por Patrones en SAP HANA SQL

🎯Operadores de Búsqueda

LIKE

Búsquedas por patrones usando comodines

Similar a "cómo"

CONTAINS

Búsquedas avanzadas de texto

Similar a "contiene"

🔤Operador LIKE

📝 Sintaxis:
expresion1 LIKE expresion2 [ESCAPE expresion3]

Función: Busca en expresion1 el patrón expresion2

% Porcentaje

Representa cero o más caracteres de cualquier tipo

_ Guión Bajo

Representa exactamente un carácter de cualquier tipo

💡Ejemplos Prácticos con LIKE

🔤 Empezar por letra

SELECT * FROM clientes WHERE nombre_compania LIKE 'D%'
Empresas que empiecen por "D"

🎯 Contener letra

SELECT * FROM clientes WHERE nombre_compania LIKE '%A%'
Empresas que contengan "A" en cualquier posición

🎪 Patrón específico

SELECT * FROM dummy WHERE 'mi gato es blanco' LIKE 'mi gat_ es blanc_'
Acepta "gato/gata" y "blanco/blanca"

🚫 Excluir patrones

WHERE campo LIKE '%A%' AND campo NOT LIKE 'A%' AND campo NOT LIKE '%A'
Contiene "A" pero no empieza ni termina con "A"
⚠️ Sensibilidad a Mayúsculas:
LIKE distingue entre mayúsculas y minúsculas. 'A%' es diferente de 'a%'

🔓Carácter ESCAPE

Para buscar los caracteres especiales % y _ como literales:

SELECT * FROM dummy WHERE 'principal.html' LIKE '%#%principal.html' ESCAPE '#'

Explicación: Usa '#' como escape para buscar el carácter '%' literal

🔧 Consejo:
Usa caracteres especiales como escape (#, @, \) para evitar conflictos con el texto normal

🚀Operador CONTAINS

📝 Sintaxis:
CONTAINS(columna, 'patron_busqueda', [nivel_fusion])

Función: Búsquedas avanzadas con múltiples capacidades
Capacidad LIKE CONTAINS
Búsqueda exacta ✅ ✅
Búsqueda aproximada ❌ ✅
Búsqueda en múltiples columnas ❌ ✅
Búsqueda por fragmentos Limitada ✅
Operadores lógicos Manual Automático

🧪Ejemplos con CONTAINS

Creamos una tabla de prueba:

CREATE COLUMN TABLE frase ( frase VARCHAR(50) ); INSERT INTO frase VALUES ('El perro ladra'), ('El gato maulla'), ('El perro ladra otra vez'), ('El perro Tommy y el gato Jerry son amigos'), ('El gato es blanco'), ('Perro, gato, perro y gato');

🔍 Búsqueda Literal

SELECT * FROM frase WHERE CONTAINS(frase, 'perro gato')
Busca la frase exacta "perro gato"

🔄 OR Lógico

SELECT * FROM frase WHERE CONTAINS(frase, 'PERRO GATO')
Busca registros que contengan "PERRO" O "GATO"
🔍 Diferencia Clave:
• Minúsculas: búsqueda literal de la frase completa
• Mayúsculas: búsqueda con OR lógico de cada palabra

📊Cuándo Usar Cada Operador

Escenario Recomendación Ejemplo
Patrones simples LIKE Nombres que empiecen por "A"
Búsquedas exactas básicas LIKE Códigos con formato específico
Búsquedas de texto avanzadas CONTAINS Búsquedas en descripciones
Múltiples términos CONTAINS Artículos con varias palabras clave
Búsquedas aproximadas CONTAINS Tolerancia a errores de escritura

🎓Resumen
📋 Puntos Clave:
• LIKE: Ideal para patrones simples con comodines % y _
• CONTAINS: Potente para búsquedas de texto avanzadas
• Ambos son sensibles a mayúsculas y minúsculas
• ESCAPE permite buscar caracteres especiales como literales
• CONTAINS se verá en profundidad en análisis de texto avanzado
